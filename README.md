# このプログラムについて

Ruby合宿2016夏のテーマである「逃避ゲーム」のプログラムです。
逃避ゲームとは、ここでは、妨害要素をうまく回避して目的の達成を目指すゲームのことです。

# 起動方法

下記のコマンドを実行してください。

```
> ruby main.rb
```

# 操作方法

・「Spc」: ジャンプします（上に移動します）
・「↓」: 下に移動します
・「→」: 右に移動します
・「←」: 左に移動します
・「ESC」: ゲームを終了します
・「D」: 隠しコマンド

# ゲームについて

## ルール

どじょう掬いまんじゅうを買うために、島根の名所をまわりながら、落ちているお金を拾うゲームです。
敵キャラクタを上手く避けながら、画面上のコインを集めて最後に購入できたまんじゅうの数を競います。
ステージは全部で3つあります。
プレイヤーは3回敵に当たると行動不能になります。

下記のいずれかを満たすと、ゲーム終了となります。

・プレイヤーが行動不能になる
・ゴールする

## 敵キャラクタ

敵キャラクタは2種類存在します。

・アジ : 三段ジャンプで移動します
・ドジョウ : 上下に移動します

敵キャラクタに当たると、ライフが1つ減ります。

##アイテム

アイテムは2種類存在します。

・しじみ : 敵が一定時間お金に変わります
・えび : 敵が一定時間お金に変わります
